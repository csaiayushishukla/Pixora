# Pixora

Pixora is a MERN stack social media application where users can upload and view images.

## Tech Stack
- React
- Node.js
- Express
- MongoDB
- ImageKit

## Features
- Upload images
- View image feed
- Backend API using Express
- Image hosting using ImageKit

## Installation

Clone the repository:

git clone https://github.com/csaiayushishukla/Pixora.git

Backend:

cd backend
npm install
node server.js

Frontend:

cd frontend
npm install
npm run dev
